## Get Started
### Installation

1. **_Clone the repository_**

```sh
git clone https://github.com/saidseghirakram/finale_devfest_website
```

2. **_Navigate to repository directory_**

```sh
$ cd finale_devfest_website
```

3. **_Install dependencies_**

```sh
$ npm install
```

### Running

1. **_Compile and hot-reload for development_**

```sh
$ npm run dev
```
